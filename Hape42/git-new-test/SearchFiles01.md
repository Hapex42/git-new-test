# Grok Prompt

Search the web for the most current building code versions in use in Nassau
County, Florida as of 2026-01-21. Identify each code family (e.g., Florida
Building Code / IBC, IRC, IECC, NFPA, etc.), the edition year, and the
effective adoption date. Prioritize official Nassau County or Florida Building
Commission sources, and include direct links to the documents or pages that
confirm the versions in force.
